**📘 Semáforo – README**

**🖥️ Sobre o Projeto**
Este é um projeto desenvolvido com HTML, CSS e JavaScript que simula o funcionamento de um semáforo real. O objetivo é treinar lógica de programação e manipulação do DOM de forma simples e visual.

**O projeto inclui:**
◦🚦 Ciclo automático das luzes(ainda vai ser aplicado)
◦🎛️ Controle manual por botões

✨ **Funcionalidade**

◦🚦 Ciclo Automático
As luzes vermelha, amarela e verde acendem em sequência com tempos definidos.(ainda vai ser aplicado)

◦🎛️ Controle Manual
Botões permitem acender cada luz individualmente.

🛠️ **Tecnologias Utilizadas**

🔹 HTML – Estrutura do semáforo e botões
🔹 CSS – Estilo, cores e posicionamento
🔹 JavaScript – Lógica de troca de luzes e temporização

▶️ Como Executar o Projeto
Abra o arquivo index.html no navegador de sua preferência.
Não é necessário instalar nenhum programa adicional — o projeto funciona 100% no front‑end.
